# Contributing to Panrise

Thanks for helping solo founders navigate global business compliance.

## What We Need

**High priority:**
- Updated tax rates, visa policies, or forex limits (things change fast)
- New nationality-specific guides (Korean, Japanese, LatAm founders need more depth)
- Company exit/closure procedures by jurisdiction
- Real-world cost corrections (if our numbers are off, tell us)

**Also welcome:**
- New jurisdiction coverage
- Country-specific accounting tool recommendations
- Translations (see `i18n/` for existing languages)
- Typo and grammar fixes

## How to Contribute

### Knowledge corrections

1. Fork the repo
2. Edit the relevant `references/*.md` file
3. Include your source (government website, official tax authority, professional experience)
4. Submit a PR with a clear title like: "Update SG corporate tax rate for 2026"

### New reference files

1. Follow the existing format — markdown with tables, headers, and practical numbers
2. Write for solo founders, not tax lawyers — plain language, specific costs, actionable steps
3. Include a "When to get professional help" section
4. Add the file to the Reference Files list in `SKILL.md`

### Translations

1. Copy `SKILL.md` to `i18n/SKILL.{lang_code}.md`
2. Translate all content — headers, body, tables, decision trees
3. Keep tool names (Stripe, Mercury, Wise), country codes, API URLs, and `references/` paths in English
4. Use natural language tone, not machine-translated stiffness

### Style guide

- Numbers over adjectives: "$160 setup cost" not "affordable setup cost"
- Trade-offs over recommendations: "cheaper but more hassle" not just "recommended"
- Honest about limits: "This is general guidance" disclaimer always present
- Revenue thresholds: always include when a recommendation applies

## Not in Scope

- Specific legal advice for individual situations
- Tax planning for corporations with employees (this is solo founder focused)
- Cryptocurrency tax optimization (too volatile, too jurisdiction-specific)
- Immigration law beyond business visa basics
